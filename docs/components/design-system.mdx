Fold Health's design system is token-driven, defined in `src/tokens/tokens.css`, and supports four themes.

## Typography

| Property | Value |
|----------|-------|
| Font family | Inter (system-ui fallback) |
| Base size | 14px |
| Base weight | 400 |
| Line height | 1.2 (global), 1.5-1.6 (body text) |
| Rendering | `-webkit-font-smoothing: antialiased` |

Weight scale: 400 (regular), 500 (medium), 600 (semibold).

## Themes

Four themes are supported via `[data-theme]` CSS selectors on `:root`:

| Theme | Primary anchor | Sidebar |
|-------|---------------|---------|
| Light | #8C5AE2 (purple) | #1A0647 |
| Dark | Brightened purple | #1A0647 |
| Blue | #007BFF | Navy |
| Plum | #6C0C46 | Deep plum |

Theme transitions animate at 200ms ease on all color and shadow properties.

## Icons

**Primary set:** Solar (Iconify) — `solar:*-linear` variants preferred.

**Stroke normalization:** A global CSS rule in `src/index.css` forces all Solar linear icons from 1.5px stroke to 1px.

**Custom icons:** Fold Health custom SVGs (filter, SMS, campaign, call history) are registered in the `Icon` component and must use `strokeWidth="1"`.

**Size convention:**

| Size | Pixels |
|------|--------|
| S | 14px |
| M | 16px |
| Default | 18px |
| L | 20px |
| XL | 24px |

## Spacing

The spacing system follows an 8px increment grid:

- **Gaps / padding:** 8px, 12px, 16px, 20px, 24px
- **Drawer padding:** 24px horizontal, 20px vertical
- **Modal padding:** 16-32px depending on content

## Sidebar chrome

The sidebar is always dark regardless of theme:

| Token | Value |
|-------|-------|
| `--sidebar-bg` | #1A0647 |
| `--sidebar-fg` | rgba(255, 255, 255, 0.85) |
| `--sidebar-active-bg` | #5020A0 |

## Chart palette

12 chart colors (`--chart-1` through `--chart-12`) are defined for data visualizations. 10 accent hues with dark and light variants are available for tags and chips.
