```
    └── AppLayout.jsx  # Root layout with sidebar + routing
```

## Conventions

- **One component per folder** with a `.jsx` file and optional `.module.css`
- **Features are self-contained** — each feature folder has its own components, styles, and data
- **Shared components** in `src/components/` are reusable primitives consumed across features
- **No barrel exports** — import directly from the component file
- **CSS Modules** for scoped styles, design tokens via CSS custom properties
