## Common controls

Every dashboard view includes:

- **Period selector** — Choose date ranges for data analysis
- **Practice filter** — Scope data to specific practices or providers
- **Drill-down** — Click into chart elements for detailed breakdowns
- **Export** — Download data and visualizations

## Charts

Built with [Recharts](https://recharts.org), the charting library provides responsive, composable chart components that follow the Fold design system tokens.
