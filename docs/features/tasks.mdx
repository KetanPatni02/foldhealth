title: Tasks
description: Task management with list view, kanban board, and drag-and-drop organization.
---

The Tasks page provides task tracking for care management workflows.

## Views

### List view

Tasks are grouped by status (Pending, Missed, Completed) with collapsible section headers. Each task row shows:

- Circular checkbox with completion state
- Task name with parent/subtask hierarchy
- Meta line (Care Journey, Automation, created by)
- Attachment and comment count badges
- Priority icons (high, medium, low)
- Status badges, due dates, assignee, and labels

### Kanban board

Drag-and-drop columns (Pending, Missed, Completed) using `@dnd-kit`. Cards move between columns with optimistic local state updates and Supabase persistence.

## Filters

Nine filter chips in the filter bar:

- Assigned to, View By, Sort By, Created By, Members, Task Status, Priority, Due Date, Labels

## Task tabs

| Tab | Description |
|-----|-------------|
| All Tasks | Complete task list |
| Assigned to Me | Tasks assigned to current user |
| My Task Pool | Available tasks to claim |
| Created by Me | Tasks the user created |
| Mentions | Tasks where the user is mentioned |

## Task detail drawer

Click a task to open the preview drawer with:

- Status dropdown (Radix Select)
- Editable detail fields
- Subtask section
- Rich text description with formatting toolbar
- Activity log with comments and history
