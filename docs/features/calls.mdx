The Calls page provides a complete view of all patient communication activity.

## Layout

- **Left panel** (350px) — Call history list with direction-coded avatars (outgoing, incoming, missed, answered, declined), search, and a dial pad
- **Main panel** — Patient profile banner, quick actions, and a call log table

## Call tabs

| Tab | Description |
|-----|-------------|
| All | Complete call history |
| Incoming | Received calls |
| Outgoing | Placed calls |
| Missed | Unanswered calls |
| Calling Agents | AI agent call activity |

## Active call features

- Real-time call monitoring with live transcript
- Call transfer flows between agents and care managers
- Call duration tracking and recording

## Call Queue Drawer

Accessed from the Agents table, the Call Queue Drawer shows three tabs:

- **Ongoing Call** — Active calls with member info, live duration, and transcript access
- **In Queue** — Queued members with priority ordering and reorder controls
- **Call Log** — Historical call records with outcomes
